# Dream-11

Simple responsive Player selection website built for an assignment.

Uses Fake Player API for show player stats.

### Pages
- Home
- Available
- Selected

### Features
- Responsive navbar
- Players Card
- Free Credit Claim (max-3times)
- Add Players
- Remove Players

### Tech
- HTML
- Tailwind CSS v4 (CDN)
- DaisyUI v5 (CDN)
- Vanilla JavaScript
- Fake Players API

### How to run
1. Clone the repo
2. Open index.html in browser (or use Live Server)
3. Use deployed site (https://dream-11-select.netlify.app/)
